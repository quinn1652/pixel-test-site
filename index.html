Pixel Test Site

Research use only. Generic test forms instrumented with a configurable Facebook pixel. Configure your pixel ID in pixel-config.js before testing.
  • Version: Dec 2019 Slick.js slider wizard · radio auto-advance · select + manual Next · checkboxes · submit
  • Version: Mar 2021 Big square buttons replace radio labels · label IS the button · SubscribedButtonClick fires on every selection · show-more pattern · same slider/select/checkbox backbone
Before testing: open pixel-config.js and set pixelId to your own Facebook test pixel ID. While dryRun: true, all fbq() calls are logged to the browser console and nothing is transmitted. Set dryRun: false to send live beacons.
